# Prompts

Prompts reutilizaveis para tarefas comuns. Copie, adapte, e envie.

| Arquivo | Quando usar |
|---------|-------------|
| [code-review.md](code-review.md) | Pedir code review rigoroso |
| [bug-fix.md](bug-fix.md) | Reportar bug e pedir fix |
| [feature-implementation.md](feature-implementation.md) | Implementar feature completa |
| [decision.md](decision.md) | Pedir decisao rapida |
| [documentation.md](documentation.md) | Pedir documentacao |
| [refactor.md](refactor.md) | Pedir refatoracao |

## Como usar

1. Copie o prompt
2. Substitua os placeholders ([SUA PERGUNTA AQUI], etc)
3. Envie no chat

## Dica

Salve seus prompts favoritos como **slash commands** em `~/.config/opencode/command/`.

Por exemplo: o prompt `code-review.md` ja eh um comando em `/review`.
