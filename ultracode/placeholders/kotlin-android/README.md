## Conteudo da Skill (`SKILL.md`)

- **Stack**: Kotlin 2.1+ (K2 compiler), Jetpack Compose BOM 2026+, Gradle KTS, Android SDK 35
- **Arquitetura**: MVVM + Clean (UI → ViewModel → UseCase → Repository → DataSource)
- **UI**: Jetpack Compose + Material Design 3, LazyColumn, theming, adaptive layouts
- **Assincronia**: Coroutines + Flow, viewModelScope, lifecycleScope, flowWithLifecycle
- **DI**: Hilt (modules, providers, singletons)
- **Persistencia**: Room (DAO, migrations, Flow queries)
- **Rede**: Retrofit + OkHttp + Kotlinx Serialization
- **Navegacao**: Navigation Compose, deep links, argumentos tipados
- **Testes**: JUnit 5 + MockK + Compose UI Test + Turbine (Flow testing)
- **Performance**: Baseline profiles, Macrobenchmark, Coil/Glide cache, DerivedStateOf
