# Testes Por Propriedades

Use em parsers, validadores, calculos ou transformacoes com grande espaco de entrada.

1. Escreva invariantes observaveis antes de gerar casos.
2. Use a biblioteca ja presente ou proponha uma dependencia antes de instala-la.
3. Inclua exemplos regressivos para falhas encontradas.
4. Execute a suite com sementes reproduziveis quando possivel.
