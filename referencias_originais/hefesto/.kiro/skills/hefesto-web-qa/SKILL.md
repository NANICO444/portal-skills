# Validacao Web Local

1. Identifique como iniciar a aplicacao e qual URL local usar.
2. Verifique o fluxo afetado, incluindo sucesso, erro e estado vazio quando aplicavel.
3. Observe console e requisicoes relevantes se a ferramenta permitir.
4. Nao trate uma inspeção visual como prova de backend sem testar o contrato.
5. Relate passos reproduziveis e defeitos encontrados.
