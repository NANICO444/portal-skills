# Plano Tecnico

O plano deve declarar:

- objetivo observavel;
- arquivos a criar ou alterar;
- comportamento esperado por etapa;
- testes ou comandos que provam o resultado;
- riscos, compatibilidade e rollback quando necessario;
- fora de escopo.

Evite arquitetura especulativa. Se faltar informacao decisiva, confirme com o usuario antes de editar.
