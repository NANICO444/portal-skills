# Desempenho

1. Defina qual lentidao ou consumo precisa ser melhorado.
2. Colete evidencia com ferramentas disponiveis no projeto ou ambiente local.
3. Altere somente o gargalo demonstrado.
4. Repita a medida depois da alteracao.
5. Separe resultado comprovado de hipotese ainda nao medida.
