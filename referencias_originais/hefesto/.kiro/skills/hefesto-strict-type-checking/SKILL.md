# Checagem Estrita De Tipos

1. Leia as configuracoes atuais antes de sugerir maior rigor.
2. Corrija tipos na area afetada pela tarefa primeiro.
3. Evite mascarar falhas com casts ou supressoes genericas.
4. Se a mudanca de configuracao ampliar o escopo, apresente o impacto antes de aplicar.
5. Rode o verificador de tipos e registre resultado.
