# Exportacao Manual De Contratos

Use quando o usuario quiser repassar informacoes tecnicas a outro workspace.

1. Gere apenas artefatos solicitados, como `CONTRATO_API.md`, `OPENAPI.yaml` ou `VALIDACAO.md`.
2. Inclua versao, entradas, saidas, erros, exemplos e pendencias.
3. Salve no local indicado pelo usuario ou proponha uma pasta de intercambio.
4. Nao abra, escreva ou execute nada em outro workspace sem pedido explicito.
