# Implementacao De Feature

- Siga padroes existentes de modulo, nomenclatura, erro e testes.
- Modele interfaces entre partes antes de duplicar logica.
- Valide entrada e trate falhas de forma observavel e segura.
- Nao adicione dependencia nova sem necessidade demonstravel.
- Atualize testes e documentacao diretamente relacionados.
- Execute verificacoes reais antes de reportar.
