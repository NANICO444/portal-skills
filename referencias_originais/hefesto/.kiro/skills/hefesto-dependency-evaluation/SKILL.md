# Avaliacao De Dependencias

- Confirme se o problema nao pode ser resolvido com stack existente.
- Verifique documentacao oficial, manutencao, licenca, seguranca e tamanho do impacto.
- Identifique custo operacional, lock-in e compatibilidade.
- Recomende adicionar somente com beneficio concreto.
- Peça confirmacao se exigir instalacao global, servico externo, custo ou credencial.
