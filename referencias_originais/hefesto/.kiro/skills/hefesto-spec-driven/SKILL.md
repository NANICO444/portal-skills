# Implementacao Guiada Por Requisitos

1. Extraia objetivo, comportamento esperado, limites e casos de erro.
2. Liste perguntas somente quando uma suposicao segura nao for possivel.
3. Transforme requisitos em criterios testaveis.
4. Implemente em etapas pequenas e valide cada criterio relevante.
