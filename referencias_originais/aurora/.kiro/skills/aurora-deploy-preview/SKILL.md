# Preview E Staging

- Prefira primeiro executar localmente e mostrar como validar.
- Identifique plataforma ja usada pelo projeto antes de sugerir nova.
- Explique o que sera enviado para servico externo e quais configuracoes seriam necessarias.
- Exija autorizacao antes de login, upload, deploy ou criacao de recursos externos.
- Nunca promover uma versao para producao sem pedido expresso.
