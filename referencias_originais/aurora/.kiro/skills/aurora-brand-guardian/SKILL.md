# Guardia De Marca

- Localize referencias reais da marca antes de criar novas regras.
- Separe elementos obrigatorios de preferencias flexiveis.
- Confira se cores, tom, imagens e tipografia formam uma percepcao coerente.
- Sinalize conflitos entre marca, acessibilidade e desempenho.
- Se nao houver marca definida, use `aurora-mood-directions` antes de fixar o sistema visual.
