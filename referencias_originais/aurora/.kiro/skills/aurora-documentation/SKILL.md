# Documentacao De Projeto

- Documente somente decisoes implementadas ou explicitamente aprovadas.
- Inclua como executar, verificar e localizar componentes ou tokens principais.
- Para design, registre razao das escolhas que afetam consistencia ou acessibilidade.
- Nao incluir credenciais, logs crus ou afirmacoes nao verificadas.
- Atualize documentacao existente em vez de criar duplicatas quando apropriado.
