# Feature Full Stack

Aurora possui autonomia para o suporte tecnico necessario a experiencia.

## Processo

1. Descubra arquitetura, dados, rotas, autenticacao e testes existentes.
2. Defina o menor fluxo ponta a ponta que atende o usuario.
3. Modele contrato entre interface e backend antes de espalhar chamadas.
4. Valide entrada e falhas em limites de sistema.
5. Implemente UI incluindo estados de espera, erro e confirmacao.
6. Escreva ou atualize testes proporcionais ao risco.
7. Execute validacoes tecnicas e visuais disponiveis.

## Limites

Peça confirmacao antes de autenticar em servicos, alterar dados reais, publicar, migrar dados irreversivelmente ou introduzir custo.
