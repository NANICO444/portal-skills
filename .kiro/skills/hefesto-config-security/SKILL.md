# Seguranca De Configuracao

1. Inspecione somente arquivos do workspace em uso.
2. Procure segredos expostos, permissoes amplas e comandos perigosos.
3. Proponha correcoes reversiveis sem exibir valores sensiveis.
4. Nao modifique configuracao global do computador sem autorizacao.
5. Informe claramente riscos confirmados e pontos apenas suspeitos.
