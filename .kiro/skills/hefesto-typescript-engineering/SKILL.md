# Engenharia TypeScript

1. Identifique framework, scripts, configuracao TypeScript e testes existentes.
2. Evite `any` e contratos implicitos quando tipos reais puderem ser expressos.
3. Trate estados de carregamento, erro e ausencia de dados.
4. Teste comportamento novo no nivel adequado.
5. Rode lint, typecheck, testes e build quando configurados.
