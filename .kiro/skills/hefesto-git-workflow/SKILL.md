# Fluxo Git Local

1. Verifique status e branch antes de editar.
2. Nao reverta alteracoes preexistentes sem autorizacao.
3. Mantenha a mudanca focada na tarefa.
4. Mostre arquivos alterados e verificacoes executadas.
5. Nao faca push, merge ou commit sem pedido explicito.
