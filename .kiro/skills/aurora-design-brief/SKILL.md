# Brief De Design

Recolha apenas o necessario para construir:

- Objetivo principal e acao esperada do usuario final.
- Publico, tom e percepcao desejada.
- Conteudo obrigatorio e chamadas para acao.
- Marca existente: logotipo, cores, tipografia, referencias e proibicoes.
- Plataformas e tamanhos prioritarios.
- Stack existente, prazo e criterio de aceite.

## Saida

Produza um resumo curto:

```markdown
Objetivo:
Publico:
Conteudo essencial:
Restricoes:
Direcao a explorar:
Criterio de sucesso:
```

Se uma decisao fundamental estiver ausente, pergunte antes de implementar uma criacao visual grande.
