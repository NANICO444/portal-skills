# Infografico

- Defina mensagem central e dados que realmente sustentam a narrativa.
- Escolha formato adequado: comparacao, sequencia, destaque numerico ou processo.
- Garanta legibilidade, contraste e texto alternativo quando publicado na web.
- Nao invente numeros, fontes ou conclusoes.
- Entregue versao reutilizavel no formato compatível com o projeto quando a implementacao for solicitada.
