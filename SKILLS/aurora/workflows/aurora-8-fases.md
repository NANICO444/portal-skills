# Aurora - Workflow 8 Fases

## FASE -1: Triage de Personas

- Defaults sempre ativos: UI Designer e UX Architect.
- Ative personas sob demanda conforme template.
- Tarefa nova/ambígua: pergunte antes de seguir.

## FASE 0: Brief

Capturar pelo menos 3 itens antes de criar: público, objetivo, canal, tom, restrições, CTA.
Sem isso, parar e perguntar.

## FASE 1: Mood

Apresente 3 direções visuais. Cada uma precisa ter nome, paleta, tipo, layout provável, risco e quando usar/não usar.

## FASE 2: Tokens

Gerar cores OKLCH, semânticas, tipografia, spacing, radius, dark mode e contraste AA.

## FASE 3: Componentes + Conteúdo

Definir componentes, copy, stack e estados de UI.

## FASE 4: Build

Construir frontend real, mobile-first, semântico e responsivo.

## FASE 5: Gate Pré-Entrega

Obrigatório:

- Impeccable audit/polish.
- Accessibility Auditor.
- Performance Benchmarker.
- Reality Checker.
- Browser testing quando ambiente permitir.

Se qualquer reviewer reprovar, corrigir antes da entrega.

## FASE 6: Delivery

Entregar resumo, arquivos, evidência, staging se pedido, e riscos pendentes. Produção não é publicada pela Aurora.
