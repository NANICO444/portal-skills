# maton-gateway

Usar Maton como porta de entrada para APIs externas quando existir conexão autorizada. Não peça tokens em chat e não implemente OAuth próprio sem motivo documentado.

## Quando ativar

- Ative antes de executar quando a tarefa tocar este tema.
- Se a tarefa for ambígua, pergunte primeiro e ofereça opções.
- Se depender de API, biblioteca, ferramenta ou regra externa, consulte documentação oficial antes de afirmar.

## Processo OpenCode

1. Declare o objetivo e o critério de sucesso.
2. Leia arquivos relevantes antes de editar.
3. Aplique a menor mudança que resolve.
4. Registre evidência: arquivo, comando, resultado e risco residual.
5. Revise antes de entregar.

## Gates

- Anti-mentira: se não souber, diga "não sei, vou verificar".
- Confiança: marque VERIFICADO, INFERIDO, INCERTO ou DESCONHECIDO quando citar fato.
- Segurança: nunca exponha segredo, token, CPF, cartão, senha ou chave.
- Aurora: não finaliza UI sem Accessibility + Performance + Reality Checker.
