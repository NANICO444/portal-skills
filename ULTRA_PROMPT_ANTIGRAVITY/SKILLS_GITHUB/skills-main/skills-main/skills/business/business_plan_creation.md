## QUANDO USAR
- Ao criar um plano de negócios para nova empresa ou expansão
- Ao buscar investimento (angel, venture capital, banco, BNDES)
- Ao estruturar uma franquia ou unidade de negócio
- Ao planejar o lançamento de um produto/serviço relevante

---

## ESTRUTURA COMPLETA DO PLANO DE NEGÓCIOS

### VISÃO GERAL DA ESTRUTURA

```
PLANO DE NEGÓCIOS — [Nome da Empresa/Projeto] — [Data]

1. SUMÁRIO EXECUTIVO              (1-2 páginas)
2. DESCRIÇÃO DA EMPRESA           (2-3 páginas)
3. ANÁLISE DE MERCADO             (3-5 páginas)
4. PLANO DE MARKETING & VENDAS    (3-4 páginas)
5. PLANO OPERACIONAL              (2-3 páginas)
6. ESTRUTURA ORGANIZACIONAL       (1-2 páginas)
7. PLANO FINANCEIRO               (5-8 páginas)
8. ANÁLISE DE RISCOS              (1-2 páginas)
9. APÊNDICES                      (conforme necessário)

TOTAL RECOMENDADO: 20-35 páginas para investidores
                   10-15 páginas para uso interno
```

## CHECKLIST DE QUALIDADE DO PLANO

**Estrutura:**
- [ ] Sumário Executivo independente e completo
- [ ] Todas as 8 seções preenchidas
- [ ] Consistência entre seções (números batem)
- [ ] Linguagem clara, sem jargão excessivo

**Mercado:**
- [ ] TAM/SAM/SOM com fontes citadas
- [ ] Análise competitiva honesta (não ignora concorrentes)
- [ ] Persona/ICP detalhada com dados reais

**Financeiro:**
- [ ] Premissas explícitas e justificadas
- [ ] DRE + Fluxo de Caixa coerentes
- [ ] Breakeven calculado
- [ ] 3 cenários modelados
- [ ] Unit economics saudáveis (LTV:CAC >3)

**Investidor-Ready (se aplicável):**
- [ ] Uso dos recursos detalhado
- [ ] Runway claramente calculado
- [ ] Valuation com metodologia explicada
- [ ] Tração atual documentada
- [ ] Time com credenciais relevantes

## DICAS PARA APRESENTAÇÃO (PITCH DECK)
Estrutura de 10-12 slides (regra de Guy Kawasaki):
1. Título / Tagline
2. O Problema
3. A Solução
4. Por que agora? (Timing)
5. Tamanho de Mercado (TAM/SAM/SOM)
6. Produto (demo/screenshots)
7. Modelo de Negócio
8. Tração / Validação
9. Concorrentes + Diferenciação
10. Time
11. Financeiro (projeções + uso dos recursos)
12. Pedido (quanto, para quê, o que ofereço)
