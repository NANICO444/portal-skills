- Ao analisar uma empresa existente (concorrente, parceiro, alvo de investimento, benchmarking)

### 6. Framework VRIO (Vantagem Competitiva Sustentável)
**Definição:** Avalia se um recurso/capacidade gera vantagem competitiva sustentável.

**Template:**
```
VRIO — [Recurso/Capacidade]

V — VALIOSO? (cria valor para o cliente ou reduz custos?)
  Sim / Não → Se Não: Desvantagem Competitiva

R — RARO? (poucos concorrentes possuem?)
  Sim / Não → Se Não: Paridade Competitiva

I — INIMITÁVEL? (difícil de copiar? barreiras históricas, sociais, complexidade?)
  Sim / Não → Se Não: Vantagem Temporária

O — ORGANIZADO? (a empresa está estruturada para explorar esse recurso?)
  Sim / Não → Se Não: Vantagem Inexplorada

RESULTADO:
- V=S, R=N, I=N, O=N → Paridade competitiva
- V=S, R=S, I=N, O=N → Vantagem temporária
- V=S, R=S, I=S, O=N → Vantagem inexplorada
- V=S, R=S, I=S, O=S → Vantagem competitiva SUSTENTÁVEL
```

### 7. Análise Financeira Básica
**Indicadores-chave:**
```
INDICADORES FINANCEIROS — [Empresa] — [Período]

LIQUIDEZ:
- Liquidez Corrente = Ativo Circulante / Passivo Circulante (ideal: >1)
- Liquidez Seca = (AC - Estoques) / PC

RENTABILIDADE:
- ROE = Lucro Líquido / Patrimônio Líquido (%)
- ROA = Lucro Líquido / Ativo Total (%)
- Margem Bruta = (Receita - CMV) / Receita (%)
- Margem Líquida = Lucro Líquido / Receita (%)
- EBITDA = Lucro + Juros + Impostos + Dep/Amort

ENDIVIDAMENTO:
- D/E = Dívida Total / Patrimônio Líquido
- Dívida Líquida / EBITDA (solvência)

CRESCIMENTO:
- CAGR Receita (3 anos) = (Receita Atual / Receita Base)^(1/n) - 1
- Crescimento YoY (%)

BENCHMARK SETORIAL:
- Comparar com 3 concorrentes diretos
- Posicionamento relativo (acima/abaixo da média)
```
