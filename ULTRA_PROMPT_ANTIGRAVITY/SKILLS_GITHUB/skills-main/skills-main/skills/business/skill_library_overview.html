Conceito

Como este sistema funciona

⚡
Skills sao modulos de conhecimento — cada arquivo .md e um bloco reutilizavel que pode ser carregado em qualquer chat especializado para dar ao assistente IA contexto profundo e atualizado. O script .py e executavel diretamente no terminal.
1

Skill como contexto

Cada arquivo .md contem conhecimento estruturado que e carregado no contexto do chat IA. Em vez de o assistente improvisar, ele consulta dados precisos.

2

Modular e combinavel

Para cada tarefa, voce seleciona as skills relevantes. Analise de empresa? Carregar business_analysis. Calcular DAS? Carregar brazil_tax_complete.

3

Sempre atualizado

As skills estao no GitHub com versionamento. Quando a legislacao muda (ex: nova tabela Simples), basta atualizar o arquivo e todos os chats futuros recebem o dado novo.

4

Script executavel

O brazil_cnae_database.py vai alem: e um programa Python que voce executa no terminal para consultar CNAE, calcular Fator-R e comparar regimes automaticamente.

5

Mentoria estruturada

O conjunto completo permite mentoria empresarial de nivel especializado: desde validar uma ideia, calcular viabilidade, escolher regime tributario ate planejar sucessao.

6

Expansivel

Novas skills podem ser adicionadas: CLT completo, ISS por municipio, equity/ESOP, importacao/exportacao. A estrutura ja esta pronta para crescer.
